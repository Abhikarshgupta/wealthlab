# ELSS Calculator Implementation Plan

## Input Fields
1. Investment Type: SIP or Lumpsum
2. Amount: Investment amount
3. Tenure: Years (min 3 years)
4. Expected Return: 14% (default)
5. Adjust for Inflation: Toggle

## Calculation Logic
- Same as Equity Calculator
- Minimum 3-year lock-in enforced
- Tax benefits under Section 80C
